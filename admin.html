프로모션 조건 추가

기본 지급 조건

지급건수 = 총 콜수 − (지급 시작 콜수 − 1). 예: 160건, 시작 141 → 20건 × 건당금액

미지급 조건

하나라도 충족하지 못하면 프로모션 전액 미지급 (거절율 초과, 수락률 미만, 총 콜수 미달, 주 근무일 미달 등)

추가 가산 조건

조건 충족 시 기본 지급액에 추가 지급 (건당 추가, 정액, 보장단가, 비율)

단순 참고 조건

지급 계산에 영향 없이 참고용으로만 표시

단가보장 · 콜수 구간

구간별 보장 단가를 설정하세요. 지급액 = max(0, 총콜수×보장단가 − 실제 배달수행금액). 수락률·근무일 등은 미지급/추가 조건에서 설정하세요.

적용할 프로모션 조건

쿠팡·배민 주정산서를 각각 선택하세요. 쿠팡에만 있으면 거절율, 배민에만 있으면 수락률 기준으로 계산하고, 양쪽 모두 있으면 쿠팡(거절율) 기준으로 적용합니다.

단가보장 조건이 있으면 배민 기사(배민 단독)에 배달처리비 정산서가 필요합니다. 파일명 기간은 배민 주정산서와 같아야 합니다.
